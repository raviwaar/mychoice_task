# Items Management System - Frontend

This is the client-side application for the Inventory Management System. It is a single-page application (SPA) built with React, Vite, and Chakra UI.

## Prerequisites

* **Node.js**
* **npm**

## Installation & Setup

1.  **Navigate to the frontend directory:**
    ```bash
    cd frontend
    ```

2.  **Install Dependencies:**
    ```bash
    npm install
    ```
3. **Create file 'frontend/.env.local' and Set your backend URL in .env.local file, example:**
     `VITE_BASE_URL=http://localhost:8000/api/v1/items/`
    

4. **Start the Development Server:**
    ```bash
    npm run dev
    ```
   
    ### Open in your browser: http://localhost:5173


5. **Build for production:**
    ```bash
    npm run dev
    ```

6. **To run the test suite:**
    ```bash
    npm run test
   ```
***
 
## Key Features
* Cursor Pagination: Seamless "Next/Previous" navigation that persists state in the URL.

* Unified Modal: Single modal component for both viewing and editing details.

* Chakra UI: Accessible and responsive component library.
