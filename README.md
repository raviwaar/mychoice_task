# Items Management System

A full-stack inventory application built with **Django REST Framework** and **React**. This project demonstrates pagination techniques (Cursor Pagination), robust state management, and automated testing.

## Tech Stack

* **Backend:** Python, Django, Django REST Framework (DRF)
* **Frontend:** React (Vite), Chakra UI, Axios
* **Database:** SQLite (Default)
* **Testing:** `APITestCase` (Django) & `Vitest` (React)
